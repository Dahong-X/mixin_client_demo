# The open source code for Home of CNB bot in mixin app and mixin blockchain

[The developer resource of Mixin Network](http://mixin-network.gitbook.io/mixin-network/)


## deps

* python2.7
* pip2.7

## pre-request
```
pip2.7 install -r requirements.txt
```

## run robot 
```
./run.sh
```
